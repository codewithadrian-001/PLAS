# PLAS Project

## Description
PLAS is a comprehensive web application for managing student performance and facilitating admin functionalities. This project includes a user-friendly interface and a robust backend to support various features such as user login, registration, data retrieval for students and admins, and performance predictions.

### Features:
- User Authentication
- Admin Dashboard
- Student Dashboard
- Performance Tracking

### Installation
1. Clone the repository.
2. Set up the database using `schema.sql`.
3. Configure the backend settings in `database.php`.
4. Open `index.html` to view the application.

### Technologies Used:
- HTML, CSS, JavaScript for frontend development.
- PHP for backend development.
- MySQL for database management.
